Privacy Policy

Our extension CalSyncMaster uses and transfers to any other app of information received from Google APIs in compliance with the Google API Services User Data Policy, including the Limited Use requirements.

This means that our extension has access to limited user data from the Google API, specifically Google Calendar data, only after obtaining clear and express consent from the user. The data is handled in accordance with Googles Limited Use requirements.

Any use of Google user data adheres to the following practices:

  • No Google user data is shared with others, except as necessary to provide and improve user features, comply with the law, or as part of a merger, acquisition, or sale of assets.
  • We dont use or transfer data for serving ads, including retargeting, personalized, or interest-based advertising.
  • We dont allow humans to read Google user data, unless we have the user's affirmative agreement, or it's necessary for security purposes (such as investigating a bug or abuse).
  • We dont use Google user data to create or augment user profiles for any purposes unrelated to the apps services, including advertising or personalization purposes.

For more information on Google's User Data Policy, please visit Google API Services User Data Policy.
